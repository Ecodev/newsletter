Known Problems
==============

There are no known problems other than this extension requires some knowledge of 
third-party tools like `fetchmail`_ and `lynx`_. It also requires the `mcrypt`_ 
PHP extension to be installed on the hosting web server. This extension may 
require some extra technical know-how to get properly installed and running.

Bugs and feature requests can be issued via the Newsletter project 
`issues`_ page on GitHub.

Commercial support
